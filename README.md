# Log Analyzer

This is a simple Python project that analyzes login activity from a sample log file.

I created this project to understand how security teams review logs and identify unusual login patterns.

## Features

- Counts successful login attempts
- Counts failed login attempts
- Displays a simple summary report
- Beginner-friendly implementation

## Example

Sample Logs:

Login Success
Login Failed
Login Failed
Login Success

Output:

Successful Logins: 2
Failed Logins: 2

- Python
